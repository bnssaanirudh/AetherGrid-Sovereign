# ADR-004: Event-Level Cascade Formulation and Leakage-Safe Splits

## Status
Proposed

## Context
Random node masking on static graphs is standard in academic node classification but creates severe data leakage in temporal cascading failure prediction (e.g., using "future" state knowledge to predict "past" failure sequences).

## Decision
- We formulate cascading failures at the event level (sequence of graph snapshots over time).
- We partition data into leakage-safe splits using temporal ordering: train on early snapshots, validate/test on subsequent snapshots.
- Masking is reserved solely for baseline benchmark comparisons or legacy smoke paths.

## Consequences
- Evaluation reflects true forecasting/prognostic capabilities during cascading failures.
- Prevents artificial performance inflation during offline validation.
