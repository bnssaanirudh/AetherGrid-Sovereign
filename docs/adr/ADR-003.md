# ADR-003: Native Complex Tensors and Phase-Generator Interface

## Status
Proposed

## Context
The proposal introduces a high-novelty Variational Quantum Circuit (VQC) phase generator that outputs phase shifts representing temporal decay and cascading progression. This requires modeling complex-valued wavefunctions or phase-shifted attributes in PyTorch.

## Decision
- We define a phase-generator interface that returns phase angles $\theta \in [-\pi, \pi]$ or complex tensors.
- When performing quantum-fuzzy fusion, we utilize PyTorch's native complex numbers (`torch.complex` or polar representation) to combine amplitude/membership with phase components: $z = r e^{i\theta}$.
- Decouple the model implementation from the specific hardware backend, using pure PyTorch complex operations as a CPU/GPU-compatible baseline.

## Consequences
- PyG layers must support propagating complex representations or polar representations of node/edge states.
- Clean contract validation for phase-generator inputs and outputs.
