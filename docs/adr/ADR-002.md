# ADR-002: IFS versus Pythagorean Fuzzy Semantics

## Status
Proposed

## Context
Intuitionistic Fuzzy Sets (IFS) enforce that the sum of membership ($\mu$) and non-membership ($\nu$) is at most 1: $\mu + \nu \leq 1$. In highly uncertain scenarios, this linear constraint can be too restrictive. Pythagorean Fuzzy Sets (PFS) relax this by requiring that the sum of squares is at most 1: $\mu^2 + \nu^2 \leq 1$.

## Decision
We will define domain state representations that support both IFS and PFS boundaries:
- Enforce linear constraint validations for IFS configurations.
- Enforce quadratic constraint validations for PFS configurations.
- Any fuzzy state record must declare its fuzzy family constraint configuration.

## Consequences
- HGT attention weight calculations can dynamically toggle or validate constraints depending on the fuzzy family selected.
- Models reject inputs that violate their specified fuzzy semantics.
